C - Comparando Textos

Time Limit: 1 second(s)
Memory Limit: 32 megabytes
  • Statement
  • All Submissions
  • Accepted Submissions
  • Download PDF
  • Submit

El Sr. Smith es un fanático de la electrónica, y se ha fabricado un pequeño robot. La Sra. Smith cree que el nuevo robot del Sr. Smith es sólo un juguete, pero con un poco de tu ayuda, el Sr. Smith va a demostrarle a la Sra. lo contrario. 

Como la Sra. siempre se queja del desorden que tiene el Sr. Smith en su despacho (y el Sr. Smith es un poco vago), el Sr. quiere hacer que su robot le organice los libros del estante por su nombre, alfabéticamente.

En lo que el Sr. Smith trabaja en el módulo de OCR para su robot, decides ayudarlo implementando un algoritmo para, dados los nombres de dos libros cualesquiera, saber cuál de ellos debe ir primero. 

El problema es que compró un módulo de memoria barato para su robot, y el Sr. calcula que el módulo de OCR solo dejará alrededor de 1KB de RAM libre, así que para ayudarlo, tendrás que programarlo en lenguaje ensamblador.


Input

2 líneas, un cadena de texto en cada una  (a y b). Ambas pueden contener espacios.


Output

-1 si a < b

0 si a == b

1 si a > b


Sample test(s)

Input
prueba prueba
Output
0
