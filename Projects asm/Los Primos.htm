B - Los Primos

Time Limit: 1 second(s)
Memory Limit: 32 megabytes
  • Statement
  • All Submissions
  • Accepted Submissions
  • Download PDF
  • Submit

Tu prima, la hija mayor de los Smith, te ha preguntado qué estudias. Tú, para impresionarla un poco, decides programar para ella un algoritmo en el lenguaje más complicado que conoces: ensamblador. 

Como no quieres complicarte mucho tampoco, decides programar un algoritmo sencillo. Al final, ella no se va a dar cuenta, lo importante es impresionarla con muchas cosas que no entienda. 

Decides enseñarle a programar en NASM el clásico algoritmo para decir si un número es primo o no. 


Input

La entrada consiste en una única línea que contiene el número (n) a comprobar.

2 < n < 2^32 - 1


Output

“true” (sin las comillas) en caso de que n sea primo.

“false” (sin las comillas también) en otro caso.


Sample test(s)

Input
43
Output
true
