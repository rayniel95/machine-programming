E - Primos otra vez

Time Limit: 1 second(s)
Memory Limit: 256 megabytes
  • Statement
  • All Submissions
  • Accepted Submissions
  • Download PDF
  • Submit
Dado un intevalo cerrado [a, b], Usted debe imprimir la cantidad de números primos en el intervalo.

Input

Línea 1: N, la cantidad de intervalos que se van a pedir.

Líneas 2..N: Dos numeros a, b tal que 1<= a <= b < 1000000


Output

Un número por cada intervalo que indique la cantidad de números primos entre a y b, separados por un cambio de linea.

Sample test(s)

Input
3 7 15 2 11 3 7
Output
3 5 3
